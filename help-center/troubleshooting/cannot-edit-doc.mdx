description: Steps to resolve issues editing documentation pages.
---

## Issue

You can't edit a documentation page after logging in.

## Common Causes

- No editing permissions
- Page is locked by another user

## Quick Fix

<Steps>
  <Step title="Check your permissions">
    Ensure you're an admin or editor for the project.
  </Step>
  <Step title="Page locked?">
    If another user is editing, wait until it's unlocked.
  </Step>
  <Step title="Refresh or log out/in">
    Reload the page or sign out and back in to refresh permissions.
  </Step>
</Steps>
