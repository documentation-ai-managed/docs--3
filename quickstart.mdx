## Prerequisites

Before you begin, ensure you have:

- A modern web browser like Chrome, Firefox, or Safari
- An email address for account verification
- Optional: Node.js installed if using the SDK for programmatic access

<Callout kind="info">

No credit card required for the free tier. Upgrade anytime from your dashboard.

</Callout>

## Sign Up and Log In

Create your account in under a minute.

<Steps>
  <Step title="Visit the Dashboard" icon="globe">
    Navigate to `https://dashboard.example.com/signup`.
  </Step>
  <Step title="Enter Details" icon="user">
    Provide your email, create a password, and verify via the confirmation link sent to your inbox.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to `https://dashboard.example.com/login` and sign in with your credentials.
  </Step>
</Steps>

## Create Your First Documentation Space

Organize your projects with dedicated spaces.

<Steps>
  <Step title="Click New Space" icon="plus">
    From the dashboard, select **New Space**.
  </Step>
  <Step title="Name and Configure" icon="settings">
    Enter a name like `My Project Docs`, add a description, and choose visibility (private or public).
  </Step>
  <Step title="Save and Enter" icon="check-circle">
    Click **Create**. You'll land in your new space's editor.
  </Step>
</Steps>

## Upload Your First Documents

Add content via UI or API.

<Tabs>
  <Tab title="Drag & Drop" icon="upload">
    In your space, drag files directly into the editor pane. Supported formats: Markdown (`.mdx`, `.md`), PDFs, images.
  </Tab>
  <Tab title="API Upload" icon="code">
    Use the REST API for automation.

    <CodeGroup tabs="JavaScript,cURL">
      ````javascript
      const formData = new FormData();
      formData.append('file', documentFile);
      formData.append('spaceId', 'your-space-id');

      const response = await fetch('https://api.example.com/v1/spaces/your-space-id/documents', {
        method: 'POST',
        headers: {
          'Authorization': `Bearer ${YOUR_API_KEY}`
        },
        body: formData
      });
      ````

      ````bash
      curl -X POST https://api.example.com/v1/spaces/your-space-id/documents \
        -H "Authorization: Bearer YOUR_API_KEY" \
        -F "file=@/path/to/document.md"
      ````
    </CodeGroup>

    <ParamField path="spaceId" param-type="string" required="true">
      Your documentation space identifier.
    </ParamField>

    <ParamField header="Authorization" param-type="string" required="true">
      Bearer token obtained from dashboard settings.
    </ParamField>
  </Tab>
</Tabs>

<Callout kind="tip">

Generate your `{API_KEY}` in **Settings > API Keys** for secure uploads.

</Callout>
